Web Development

For learning the basics of CSS, HTML, and Javascript, we recommend checking out online resources like w3schools or Free Code Camp. Other useful things to learn include frameworks for handling JSON data like JQuery, Moustache, and Angular.

If you’re looking to skip styling your website or are looking for re-useable interfaces, check out Bootstrap or Material Design Lite.

If you’re looking for something more advanced, check out awesome tools and frameworks like React for frontend, Node.js, Flask, or Django for backend. Other awesome frameworks exist! Don’t be afraid to Google for the tools that will meet your needs.

Often, when dealing with backend, you’ll also need a database. SQL and MongoDB are popular languages to learn, and you can find tutorials for these at the links in the first paragraph of this section. Another popular database is Google’s Firebase, which has a free tier if you want to experiment with data.

Lastly, if you’re looking to deploy your website, try out Github Pages (for static sites only), Heroku, or Netifly. GitHub pages also is connected with Jekyll, making it easy for you to deploy your personal site or a project site using only markdown!


Mobile Development

There are a few different ways to develop mobile apps! If you're interested in developing for both Android and iOS, check out Flutter, React Native, or Kotlin. For all of these frameworks/languages, you can build native applications for both platforms without having to write in two different languages. Pretty convenient!

However, if you're looking to build something for just one platform, you'll have to look at some more specific tools. For iOS, you will need to develop in Swift. The unfortunate barrier to this is that you must have an Apple device that runs XCode. If you're looking to work on Android applications, you can code in Java using Android Studio, although it seems like the trend in Android to heading towards Kotlin and eventually Flutter.

For deploying your app, keep in mind that the iOS system has a more rigorous review process and requires that you join their Apple Developer Program, which costs roughly $100. The Android stores tend to be more lenient and usually don't require the paid subscription.


Hardware

Software is fun, but hardware is cool, too! If you're interested in trying out hardware, make sure to check out the ECE Supply Center, which is open to all faculty, staff, and students and has a variety of hardware that you might be interested in. Also, the ECE Open Lab hosts tools and workspaces that you might find helpful as you explore more hardware! You will need to register and go through training in order to enter the lab.

Some cool tech that lie at the intersection of hardware and software include Raspberry Pis and Arduinos, which are affordable, powerful single-board devices that can be connected to a variety of hardware. Check out their websites for tutorials and project ideas!


Data Science and Visualization

Data science and visualization is the art of taking data and turning it into a form that is easy to understand. Some commonly used languages for data science include Python and R, although other languages are used.

For data visualization, there are many packages in Python that can help with that, including Plotly and Matplotlib. If you are more comfortable in Javascript, D3.js is a powerful tool for making visualizations.


Artificial Intelligence and Machine Learning

Artificial intelligence is a fast growing and popular field in computer science. There are a lot of great tutorials and resources online, including Towards Data Science, among others. If you're specifically interested in machine learning, there are a lot of frameworks available. They are mostly based in Python, although there are others that exist in other languages. Some popular ones include PyTorch and Tensorflow for neural networks and Scikit Learn for many other algorithms. All of these websites also have excellent tutorials that walk you through the applied basics. For more structured courses outside of UIUC, check out Coursera, MIT OpenCourseWare, or edX.
